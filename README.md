# Instructions pour compiler et lancer le projet

1. Installez Raylib sur votre système (voir https://github.com/raysan5/raylib/wiki/Working-on-GNU-Linux).
2. Compilez le projet :
   make
3. Lancez le jeu :
   make run
4. Nettoyez les fichiers générés :
   make clean

Le code source principal se trouve dans src/main.cpp.

---

Ce projet est une base pour un jeu FTL-like en C++ avec Raylib. Ajoutez vos fichiers source dans le dossier src/.
